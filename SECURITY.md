# Security Policy

## Supported Versions

KubeNAS is pre-1.0. Security fixes are provided for the latest main branch and latest tagged pre-release.

## Reporting a Vulnerability

Please report vulnerabilities privately by opening a GitHub Security Advisory draft in this repository.

Include:

- affected components
- reproduction details
- impact assessment
- suggested mitigation (if available)

We aim to acknowledge reports within 72 hours.
